# Focus Coffee | STAY FOCUS

A premium, dark-themed coffee shop landing page built with **Next.js**, **Tailwind CSS**, and **Framer Motion**.
Inspired by the **Supanova Design Skill** for high-agency frontend aesthetics.

## Concept
**Focus** is more than just a coffee shop. It's a sanctuary for those who value the essence of things. In a world full of noise, we provide a space and a taste that allow you to quiet the distractions and immerse yourself in what truly matters.

## Key Features
- **Premium Dark Aesthetic**: A deep, moody atmosphere that emphasizes focus and tranquility.
- **Micro-Animations**: Smooth reveal transitions using `framer-motion` for a refined user experience.
- **Double-Bezel Design**: Sophisticated card components with multi-layered borders and glassmorphism.
- **Optimized for Korean**: Uses the **Pretendard** font with `word-break: keep-all` for perfect typography.
- **Responsive Layout**: Designed to look stunning on all devices.

## Tech Stack
- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: Lucide React & Iconify Solar
- **Fonts**: Pretendard (Self-hosted via CDN)

## Getting Started
To run the project locally:

```bash
npm install
npm run dev
```

Build for production:
```bash
npm run build
```
